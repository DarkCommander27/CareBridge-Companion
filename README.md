# Therapy Companion

## Project Overview

The Therapy Companion is a digital tool designed to assist individuals in managing their mental health and therapy journeys. The application provides resources, tracking features, and a supportive community to enhance the therapy experience.

## Features
- **Mood Tracking**: Users can log their daily mood and experiences.
- **Resource Library**: Access to articles, videos, and exercises for mental health.
- **Goal Setting**: Users can set personal goals related to their mental health.
- **Community Support**: A forum for users to share experiences and support each other.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DarkCommander27/therapy-companion.git
   ```

2. Navigate to the project directory:
   ```bash
   cd therapy-companion
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

## Usage

After starting the application, users can:
- Create an account or log in.
- Navigate through mood tracking, resources, and goal setting features.
- Participate in community discussions.
